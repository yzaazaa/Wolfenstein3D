# WOLFENSTEIN 3D

The first fps 3d game.

## Installation

```bash
git clone https://github.com/yzaazaa/Wolfenstein3D && cd Wolfenstein3D
```

## Compiling

Making the project.

```bash
make
```

or

```bash
make bonus
```

## Running

```bash
./cub3D path_to_map
```

or

```bash
./cub3D_bonus path_to_map
```

## Cleaning the folder from object files

```bash
make clean
```

## Cleaning the project entirely

```bash
make fclean
```
