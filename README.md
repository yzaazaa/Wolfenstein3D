# WOLFEENSTEIN 3D

The first fps 3d game.

## Installation

```bash
git clone https://github.com/yzaazaa/Wolfenstein3D && cd Wolfenstein3D
```

## Compiling

Making the project.

```bash
make
```
## Running

```bash
./cub3d path_to_map
```

## Cleaning the folder from object files

```bash
make clean
```

## Cleaning the project entirely

```bash
make fclean
```
